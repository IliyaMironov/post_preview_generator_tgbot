# Шрифты

Эта папка предназначена для пользовательских шрифтов.

## Как добавить шрифт

1. Поместите файлы шрифтов (.ttf) в эту папку
2. Рекомендуемые шрифты:
   - `Arial.ttf` - основной шрифт
   - `Arial-Bold.ttf` - жирный шрифт для заголовков

## Поддержка кириллицы

Убедитесь, что выбранные шрифты поддерживают кириллицу!

## Fallback

Если пользовательские шрифты не найдены, бот будет использовать системные шрифты:
- Windows: arial.ttf, arialbd.ttf
- Pillow default font (крайний случай)
